# Mobile assets

Replace the placeholders in this directory with final brand assets before a store build.

Required files (Expo defaults will be used if missing):

| File | Size | Purpose |
|---|---|---|
| `icon.png` | 1024×1024 PNG, opaque | App icon (iOS + Android fallback) |
| `adaptive-icon.png` | 1024×1024 PNG, transparent | Android adaptive icon foreground |
| `splash.png` | 2732×2732 PNG | Splash screen (centered logo on dark background) |
| `notification-icon.png` | 96×96 monochrome PNG | Android notification icon |
| `favicon.png` | 48×48 PNG | Expo Web favicon |

Design tokens (mirror web):

- Primary / accent: `#F26522` (mcaforo orange)
- Dark surface:    `#0B1220`
- Logo:            lockup from `/public` on the web app
